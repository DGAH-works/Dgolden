# 金装武将

这是太阳神三国杀的一个武将扩展包，以弱将加强为主题，选取了原版三国杀中单挑能力不足或在多人身份局中不易发挥作用的十名武将重制而成。所含武将如下：

1. 夏侯惇（刚烈）

2. 许褚（裸衣）

3. 张辽（突袭）

4. 刘备（仁德、激将）

5. 张飞（咆哮）

6. 赵云（龙胆）

7. 魏延（狂骨）

8. 黄盖（苦肉）

9. 陆逊（谦逊、联营）

10. 关羽（武神、武魂）
